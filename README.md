# Highlight-generation-system---cricket
This project focuses on building an intelligent system that automatically generates cricket match highlights by analyzing ball-by-ball data. The system identifies key events such as boundaries, wickets, milestones, and high-impact moments to produce a concise and engaging summary of the match.

## Overview

This project automatically converts a full cricket match into a 10-minute highlight video using embedded systems, networking, and intelligent event detection.

---

## Key Components

- Embedded cameras and microphones  
- Cloud-based processing  
- AI-based event detection  
- Networking protocols (TCP, UDP, HTTP)  

---

## Highlight Events

- Wickets  
- Boundaries (4s, 6s)  
- Milestones  
- Match-winning moments  

---

## Technologies Used

- Networking (TCP/UDP, RTSP)  
- Cloud Computing  
- Video Processing  
- AI Concepts
